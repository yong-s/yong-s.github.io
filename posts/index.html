Docker更换默认目录

查询目前已安装的路径 sudo docker info | grep "Docker Root Dir" 备份旧数据到新目录 cp -a /var/lib/docker /path/to/docker 修改配置文件 docker 版本 > 17.05 vim /etc/docker/daemon.json { "data-root": "/path/to/docker", "storage-driver": "overlay2" # 这个是Docker的默认设置，这里也可以不用加 } 重启docker sudo systemctl restart docker 参考 两种方式迁移Docker
